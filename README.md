# NeonExam

![NeonExam Logo](public/logo.png)

NeonExam is an AI-powered online examination platform designed to streamline the assessment process. This repository contains the Next.js frontend application.

## Features

- **AI Question Generation:** Automatically generate exams from any reference material.
- **Secure Proctoring:** AI-driven webcam and microphone checks.
- **Dynamic Grading:** Real-time feedback and intelligent log tracking.
- **Responsive Design:** Beautiful, fast, and accessible on all devices.

## Tech Stack

- **Framework:** [Next.js](https://nextjs.org/) (App Router)
- **Styling:** Custom CSS with modern glassmorphism UI.
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Icons:** Custom SVG and CSS icons.
